## 最快上手

```bash
bash scripts/install-local.sh
bash scripts/install-opencode-global.sh
```

然后你可以在任意目录打开：

```bash
opencode
```

并直接使用：

- `/experiment-init`
- `/experiment-status`
- `/research-context`
- `/innovate-loop`
